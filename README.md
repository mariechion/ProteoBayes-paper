This repository contains all data and code necessary to reproduce the results presented in the paper: 
"A Bayesian Framework for Multivariate Differential Analysis" by Marie Chion and Arthur Leroy (2025). 
The article can be accessed at: https://arxiv.org/abs/2307.08975. 
The associated R package [ProteoBayes](https://mariechion.github.io/ProteoBayes/) is available on CRAN and on GitHub at [mariechion/ProteoBayes](https://github.com/mariechion/ProteoBayes).

## Repository Structure
- `Data/`: Contains datasets used in the analysis, including simulated and real-world data.
- `R/`: Contains R scripts for data preprocessing, experiments, and reproducing figures.
- `Results/`: Contains experiment results tables generated from the analysis.
- `Figures/`: Contains figures generated from the analysis.
